About This Project

A full-stack data science project built to demonstrate end-to-end ML engineering on a real business problem.

🎓 Project Overview

Retento.ai is an enterprise-grade churn intelligence system. It provides high-fidelity predictions and actionable retention playbooks using gradient-boosted decision ensembles and interpretable AI techniques.

Objective

Identify at-risk customers before they leave, enabling the business to intervene proactively with personalised retention offers — reducing churn rate and increasing LTV.

🧠 ML Pipeline
1
Data Preprocessing
Encoding, null handling, feature selection on 5000-row Telco CSV
2
Model Training
Logistic Regression, Random Forest, Gradient Boosting compared on ROC-AUC
3
Auto-Selection
Best model saved to disk as .pkl with full metadata JSON
4
Flask REST API
